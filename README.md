## 命令
| CLI 命令 | 功能 |
| --- | --- |
| dev-cli eslint | 对当前项目进行 Eslint 校验 |
| dev-cli webpack | 使用 Webapck 当前项目进行构建  |
| dev-cli rollup | 使用 Rollup 当前项目进行构建  |
| dev-cli git init | 本地初始化 git 项目（当前支持 GitLab 部分功能）  |
| dev-cli add tpl | 自定义添加模板 |
| dev-cli init tpl | 将添加的模板初始化到本地 |
